Engineering

Architecture Highlights

Shared design patterns across both frameworks. Built for production reliability and scale.

🧱

Modular Architecture

Core / Exporters / Templates separated. Add new scrapers without touching framework code. Open-closed principle.

🔄

Retry & Backoff

Exponential backoff with jitter. Auto-retry on 429, 5xx, timeouts, and proxy failures. Configurable max attempts.

🛡️

Anti-Detection

User-Agent rotation, proxy rotation, random delays, Playwright stealth mode. Bypass basic bot detection.

💾

Multi-Format Export

CSV (UTF-8 BOM), JSON (with metadata), Excel (styled, auto-filter), SQLite (auto schema) — all in one run.

⏰

Scheduled Scraping

Cron-like scheduler with run history JSON. Daily, hourly, weekly, or custom interval. Graceful shutdown.

📝

Production Logging

Rich-formatted console + rotating file logs. Per-module loggers. Structured output for debugging.

📄

YAML-Driven Config

CSS selectors in YAML files. Adapt to any website without code changes. Field-level regex & validation.

🔌

Template Method Pattern

BaseScraper defines lifecycle: initialize → fetch → parse → validate → export → cleanup. Subclasses implement hooks.
