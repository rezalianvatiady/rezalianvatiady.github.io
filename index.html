Shared Architecture Highlights

Modular Architecture

Core / Exporters / Templates separated. Add new scrapers without touching framework code.

Retry & Backoff

Exponential backoff with jitter. Auto-retry on 429, 5xx, timeouts, and proxy failures.

Anti-Detection

User-Agent rotation, proxy rotation, random delays, Playwright stealth mode.

Multi-Format Export

CSV (UTF-8 BOM), JSON (with metadata), Excel (styled), SQLite (auto schema) — simultaneously.

Scheduled Scraping

Cron-like scheduler with run history JSON. Daily, hourly, weekly, or custom interval.

Production Logging

Rich-formatted console + rotating file logs. Per-module loggers under 'odis' / 'wst' namespace.
